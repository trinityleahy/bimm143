Cool! This is my working snippet that I can turn into a function called grade().

All functions in R have at least three things: - name (grade) - input arguments, (student1, etc) - body, our working snippet

grade <- function(x){
  #Map/Replace NA values to 0
  x[is.na(x)] <- 0

  #Exclude the lowest score and calculate the mean
  mean(x[-which.min(x)])
  }
